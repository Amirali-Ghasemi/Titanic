# 🚢 Titanic - Machine Learning from Disaster

This repository contains an end-to-end Machine Learning pipeline to solve the classic Titanic survival prediction challenge from Kaggle. The project focuses on robust Feature Engineering, exploratory data analysis (EDA), and ensemble modeling.

## 📊 Project Structure
```text
titanic-survivor-prediction/
│
├── data/                    # Contains train.csv and test.csv from Kaggle
├── notebooks/               # Jupyter notebook containing training code
├── models/                  # Saved serialised model checkpoint (joblib)
├── README.md                # Project documentation
└── requirements.txt         # Required Python packages
